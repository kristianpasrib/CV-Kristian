Portfolio

Human Development Index (HDI) in Indonesia 2024: Data Mining and Visualization with R

R R Markdown Data Analysis

Human Development Index (HDI) in Indonesia 2024 is a project of data mining and visualization achieved using R and knitted using R Markdown, with an interactive HTML report being the end product. The aim of this project is to explore and describe Indonesian regions' most determining factors of HDI, as well as its trends of HDI development over the past five years according to interactive, informative, and easy-to-grasp data visualizations.

Suntik.In: Cloud Based Immunization Management System AWS

AWS Services Cloud Architecture Big Data

Suntik.In is a big data project showcasing a cloud architecture on AWS for efficient and secure immunization data management. It uses Lambda, API Gateway, RDS, S3, Cognito, and CloudFront to enable real-time processing, secure access, and monitoring. The final output is a detailed architecture diagram illustrating interactions between healthcare workers, administrators, and external organizations.

Bank Marketing Campaign Performance Analysis: Data Mining and Visualization Final Exam

R R Markdown Data Analysis

This project evaluates the performance of a bank's recent marketing campaigns using data-driven analysis with the R programming language. The analysis includes identifying and addressing data quality issues, exploring relationships between variables, and presenting insights through interactive visualizations. The goal is to uncover key patterns and trends, provide clear interpretations, and deliver actionable recommendations to support future marketing strategies.

BLYNC: Design and Development of a Braille Keyboard Application with Voice Feedback for Visually Impaired Users

Software Engineering HTML CSS JavaScript

BLYNC is a mobile app developed using Prototyping SDLC to help visually impaired users type faster with a Braille touchscreen keyboard. The project involved iterative prototyping, frontend development with HTML, CSS, and JavaScript, and a Flask backend for data processing. Real-time voice feedback was integrated to verify input and enhance usability, demonstrating a structured Software Engineering approach from requirement analysis to refinement.

SoilVision: Intelligent Soil Monitoring and Crop Recommendations Application Using Random Forest

Python Machine Learning Artificial Intelligent IoT

Smart Soil Advisor is an IoT and AI-based application that monitors soil conditions in real-time. Sensors collect data such as soil moisture, pH, and nutrient content, which is then analyzed using Random Forest to provide recommendations for the most suitable crops for the land. This project emphasizes the application of AI in delivering data-driven agricultural solutions through an easy-to-use application.

Cinema Ticketing System: A Movie Ticket Booking Application Based on OOP and SQL

Java SQL

A cinema ticket booking application developed using Object-Oriented Programming (OOP) and integrated with an SQL database for efficient data management. It features an intuitive Graphical User Interface (GUI) with four main menus for ticket printing, movie data, scheduling, and user management. The project applies OOP concepts such as encapsulation, inheritance, and polymorphism with SQL for structured data handling.

Simple Ticketing System using C Language

C

This program is a simple application developed in C language to manage ticket sales. It provides essential features such as adding, reading, updating, and deleting transaction data (CRUD), along with search and sorting functions to make data management more efficient. Enhancing Bone Anatomy Learning with an Augmented Reality Application Featuring Interactive Quizzes

Augmented Reality Assemblr Blender

Bone Anatomy Learning is a web-based Augmented Reality (AR) learning project developed using Assemblr and Blender, with interactive 3D visualizations as the end product. The aim of this project is to allow students to explore human anatomical structures in an engaging and immersive way, enhance their motivation and understanding, and provide interactive quizzes to test their knowledge, complementing traditional anatomy learning methods.
